# Module 09 - Git

## Overview
This module covers the fundamental concepts and operations of Git version control. The hands-on exercises demonstrate Git repository initialization, configuration, branching, merging, conflict resolution, and repository maintenance.

## Exercises Completed

### Exercise 01 - Git Configuration
- Installed and configured Git.
- Configured global username and email.
- Initialized a Git repository.
- Created and committed a sample text file.
- Verified commit history.

### Exercise 02 - Git Ignore
- Created a `.gitignore` file.
- Ignored log and temporary files.
- Verified ignored files were not tracked by Git.

### Exercise 03 - Branching and Merging
- Created a new branch.
- Switched between branches.
- Added changes in the feature branch.
- Merged the feature branch into the main branch.
- Deleted the merged branch.

### Exercise 04 - Merge Conflict
- Created conflicting changes in two branches.
- Observed merge conflicts.
- Resolved conflicts manually.
- Committed the resolved merge.
- Created and committed a `.gitignore` file.

### Exercise 05 - Git Repository Cleanup
- Verified repository status.
- Listed all branches.
- Pulled the latest changes from the remote repository.
- Pushed completed exercises to GitHub.
- Verified successful synchronization with the remote repository.
